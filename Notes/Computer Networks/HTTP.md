# HTTP

## 简述

### URI

- Uniform Resource Identifier 统一资源标识符。
- 用于标识某一互联网资源名称的字符串。
- 最常见的形式是统一资源定位符URL(如网址)，更罕见的用法是统一资源名称URN(如书籍ISBN编号)。

### 报文

- 请求报文：请求行(request line)、请求头部(header)、空行、请求数据。

  ![](https://pic002.cnblogs.com/images/2012/426620/2012072810301161.png)

- 响应报文：状态行、消息报头、响应正文。
